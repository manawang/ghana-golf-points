# 🚀 快速上手指南

## 安装与启动

### 方式一：使用启动脚本（推荐）

**Mac/Linux:**
```bash
cd golf-points-app
./start.sh
```

**Windows:**
```cmd
cd golf-points-app
start.bat
```

### 方式二：手动运行

```bash
cd golf-points-app
pip install -r requirements.txt
streamlit run app.py
```

应用将在浏览器中打开（默认 http://localhost:8501）

## 📊 使用流程

### 第一步：准备 GolfLive 数据

从 GolfLive 导出比赛结果，确保文件包含以下列：

| 列名（支持多种写法） | 说明 | 必需 |
|---------------------|------|------|
| 姓名 / Name / Player | 球员姓名 | ✅ |
| 总杆 / Gross / Total | 总杆成绩 | ✅ |
| 净杆 / Net | 净杆成绩 | 建议 |
| 差点 / HCP / Handicap | 差点 | 可选 |

**示例 CSV 格式：**
```csv
姓名,总杆,净杆,差点
张三,82,70.5,11.5
李四,85,71.0,14.0
王五,88,72.5,15.5
```

> 📁 项目中包含一个示例文件：`data/sample_golflive_export.csv`

---

### 第二步：导入比赛结果

1. 在浏览器中打开应用
2. 点击左侧菜单「📤 导入比赛结果」
3. 上传 GolfLive 导出的 CSV/Excel 文件
4. 检查数据预览是否正确
5. 点击「💾 计算积分」进入下一步

---

### 第三步：计算并保存积分

1. 填写赛事信息：
   - 📅 比赛日期
   - 🏷️ 赛事名称（如「3月月例赛」）
   - 🏆 赛事类型（周例赛/月度大赛）
   - ⭐ 是否为特殊赛事（队长杯/年终大赛）
   - ⛳ 比赛球场（可选）

2. 点击「🚀 开始计算积分」

3. 查看计算结果：
   - 🥇 总杆冠军会高亮显示（金色背景）
   - 🥇 净杆第1名显示绿色背景
   - 表格显示详细的积分 breakdown

4. 确认无误后，点击「✅ 确认并保存赛事结果」

---

### 第四步：查看积分榜

1. 点击左侧菜单「🏆 积分榜」
2. 查看 TOP 10 排行榜
3. 使用搜索功能查找特定球员
4. 可导出 CSV 文件

## 📋 积分规则速查

### 周例赛
| 排名 | 积分 |
|------|------|
| 第1名 | 30分 |
| 第2名 | 20分 |
| 第3名 | 10分 |
| 其他 | 0分 |

### 月度大赛
**全员得分，按净杆排名：**

| 排名 | 积分 | 排名 | 积分 |
|------|------|------|------|
| 第1名 | 100分 | 第6名 | 41分 |
| 第2名 | 70分 | 第7名 | 37分 |
| 第3名 | 60分 | 第8名 | 34分 |
| 第4名 | 52分 | 第9名 | 31分 |
| 第5名 | 46分 | 第10名 | 28分 |

- 第11-15名：26→18分
- 第16-25名：17→10分
- 第26-50名：9→5分（最低参与分）

**总杆冠军额外 +30分**（确保全场最高分）

### 特殊赛事（×2）
- 队长杯（Captain's Prize）
- 年终月度大赛（Year-end Monthly Major）

## 💡 常见问题

### Q: GolfLive 导出的是什么格式？
A: 通常是 CSV 或 Excel 格式。如果是 Excel，建议另存为 CSV 后再导入，兼容性更好。

### Q: 导入时提示无法识别列名怎么办？
A: 确保文件包含「姓名」和「总杆」列。系统会自动识别多种列名写法（中英文均可）。

### Q: 没有净杆数据可以计算吗？
A: 可以。如果有「差点」列，系统会自动计算净杆（总杆 - 差点）。建议尽量提供净杆或差点数据。

### Q: 并列排名怎么处理？
A: 周例赛：并列时平分该位置的积分。
例如：两人并列第1，则各得 (30+20)/2 = 25分

### Q: 数据保存在哪里？
A: 数据保存在 `data/` 目录下的 JSON 文件中：
- `events.json` - 赛事记录
- `rankings.json` - 积分排名
- `players.json` - 球员信息

### Q: 如何备份数据？
A: 进入「⚙️ 数据管理」页面，点击「📤 导出数据」按钮下载 JSON 备份文件。

### Q: 如何删除错误的赛事记录？
A: 进入「📋 赛事记录」页面，找到要删除的赛事，点击「🗑️ 删除此赛事」按钮。
