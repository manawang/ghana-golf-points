# 加纳中华高球队积分系统

一个基于 Streamlit 的 Web 应用，用于自动计算球队积分，支持直接导入 GolfLive 导出的比赛结果。

## 功能特点

- 📊 **自动计算积分**：根据球队规则自动计算周例赛和月度大赛积分
- 📥 **导入 GolfLive 数据**：直接导入 CSV/Excel 格式的比赛结果
- 🏆 **特殊赛事支持**：队长杯和年终大赛自动双倍积分
- 📈 **实时排行榜**：查看年度总积分排名
- 🎯 **差点管理**：支持 GHIN 差点系统
- 📤 **导出功能**：支持导出积分报表

## 安装与运行

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行应用

```bash
streamlit run app.py
```

应用将在浏览器中打开（默认 http://localhost:8501）

## GolfLive 数据导入格式

应用支持以下列名的 CSV/Excel 文件：

| 列名 | 说明 | 必需 |
|------|------|------|
| 姓名 / Name / Player | 球员姓名 | ✅ |
| 总杆 / Gross / Total | 总杆成绩 | ✅ |
| 净杆 / Net | 净杆成绩 | ✅ |
| 差点 / HCP / Handicap | 差点（用于计算净杆） | 可选 |

## 积分规则

### 周例赛
| 净杆排名 | 积分 |
|---------|------|
| 第1名 | 30 |
| 第2名 | 20 |
| 第3名 | 10 |
| 其他 | 0 |

### 月度大赛
- **总杆冠军**：净杆积分 + 30分（确保全场最高）
- **净杆排名积分**：
  - 第1名：100分
  - 第2名：70分
  - 第3名：60分
  - 第4名：52分
  - 第5名：46分
  - 第6名：41分
  - 第7名：37分
  - 第8名：34分
  - 第9名：31分
  - 第10名：28分
  - 第11-15名：26→18分
  - 第16-25名：17→10分
  - 第26-50名：9→5分

### 特殊赛事（×2）
- 队长杯（Captain's Prize）
- 年终月度大赛

## 使用流程

1. **准备数据**：从 GolfLive 导出比赛结果为 CSV/Excel
2. **导入数据**：在应用中上传文件
3. **设置参数**：选择赛事类型（周例赛/月度大赛）、是否为特殊赛事
4. **计算积分**：系统自动计算并显示结果
5. **保存记录**：将结果保存到数据库
6. **查看排行**：查看年度总积分排行榜

## 项目结构

```
└── data/                  # 数据存储目录
    ├── players.json       # 球员信息
    ├── events.json        # 赛事记录
    └── rankings.json      # 积分排名
```

## 注意事项

- 首次运行时会自动创建数据目录
- 支持并列排名，积分会平均分配
- 数据会自动保存，可随时查看历史记录
